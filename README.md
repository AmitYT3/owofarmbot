## ❗・Important

it can detect virus due to captcha checker please turn off your antivirus

## 👑・Features

-   Auto Hunt
-   Auto Battle
-   Inventory Check
    -   auto gem use (beta)
    -   auto lootbox use
    -   auto crate use
    -   auto eventbox use (like anniversary present or valentine's day)
-   Auto Sell All
-   Auto Pray
-   Captcha(Ban) Protection v0.0.6 beta
-   Discord-RPC
-   **Extra Token**
    -   All Main Token Features
    -   Auto Pray for Main Token

## ⚙・config.json example

```
{
    "settings": {
        "huntandbattle": "true",
        "banbypass": "true",
        "discordrpc": "true",
        "inventory": {
            "inventorycheck": "true",
            "gemcheck": "true",
            "lootboxcheck": "true",
            "cratecheck": "true",
            "eventcheck": "true"
        },
        "sell": "true",
        "pray": "false",
        "extratoken": "true"
    },
    "main": {
        "token": "",
        "userid": "",
        "channelid": "",
        "owodmchannelid": ""
    },
    "extra": {
        "token": "",
        "userid": "",
        "channelid": "",
        "owodmchannelid": ""
    }
}


```
